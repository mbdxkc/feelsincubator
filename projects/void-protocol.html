Void Protocol

Category
UI/UX
Year
2024
Role
Product Designer

Void Protocol is a decentralized communication platform designed for privacy-conscious users. The interface balances complex functionality with intuitive interactions.

The dark-first design language uses depth and glow effects to create hierarchy without relying on traditional card patterns. Every interaction provides feedback through subtle animations.

The component system was built for scalability, with tokens defining every color, spacing, and motion value. This enabled rapid iteration across web and mobile platforms.
